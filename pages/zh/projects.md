# 项目

这一页基于 2026 年 4 月 3 日 GitHub 公开仓库页同步更新。这里既包含自有仓库，也包含能够帮助解释当前公开技术方向的协作型仓库。

## 核心代表仓库

- [paper-list](https://github.com/isLinXu/paper-list)：一个公开的论文跟踪与发布工作流仓库，主要围绕收集、整理和持续更新展开
- [crablet](https://github.com/isLinXu/crablet)：自有 AI Assistant 仓库，体现了当前对 Agent 工作流和实用工具链的兴趣
- [claude-code-book](https://github.com/isLinXu/claude-code-book)：自有仓库，主要围绕编码代理工作流、使用笔记和相关工程资料
- [darwin-agent](https://github.com/isLinXu/darwin-agent)：自有仓库，和当前 Agent 系统兴趣方向一致
- [eval-skills](https://github.com/isLinXu/eval-skills)：自有评测型仓库，和技能体系、检查与质量验证相关
- [YOLOv8_Efficient](https://github.com/isLinXu/YOLOv8_Efficient)：自有计算机视觉仓库，主要围绕 YOLOv8 的训练与评测流程
- [isLinXu.github.io](https://github.com/isLinXu/isLinXu.github.io)：当前这个个人主页本身

## 贡献与协作方向

下面这些条目保留在这里，是因为它们能帮助解释当前公开方向，即使对应的公开仓库本身是 fork 形式。

- [YOLO-Master](https://github.com/isLinXu/YOLO-Master)：一个公开可见的 fork 形式工作仓库，对应 YOLO-Master 这条检测研究方向
- [LlamaFactory](https://github.com/isLinXu/LlamaFactory)：一个公开可见的 fork 形式贡献方向，和 LLM / VLM 微调工程密切相关
- MME 相关工作：Xu Lin 出现在 MME Benchmark 作者列表中，因此多模态评测是公开资料里可核对的一部分
- [claude-code](https://github.com/isLinXu/claude-code)：围绕 Claude Code 的公开 fork 形式工程版本
- [claude-code-src](https://github.com/isLinXu/claude-code-src)：面向源码与工作流理解的公开 fork 仓库
- [ms-agent](https://github.com/isLinXu/ms-agent)：与轻量 Agent 执行框架相关的公开 fork 方向
- [MobileAgent](https://github.com/isLinXu/MobileAgent)：与 GUI Agent / Mobile Agent 相关的公开 fork 仓库
- [mcore-bridge](https://github.com/isLinXu/mcore-bridge)：和 Megatron-Core 及大模型训练基础设施相关的公开 fork 方向

## 项目特点

把这些仓库放在一起看，当前公开项目主要集中在这些方向：

- 多模态 AI、LLM 与 Agent 相关工作流
- 实用工程与开发者工具
- 模型评测、技能体系与自动化实验
- 计算机视觉与 YOLO 相关应用工作
- Benchmark 与评测思路
- 公开整理、论文收集与学习档案
