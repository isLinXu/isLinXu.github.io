# 文档

文档是这个主页的重要组成部分，因为公开工作里不仅有代码，还有大量笔记、整理和学习记录。

## 代表性文档仓库

- [isLinXu.github.io](https://github.com/isLinXu/isLinXu.github.io)
- [paper-read-notes](https://github.com/isLinXu/paper-read-notes)
- [awesome-road-map](https://github.com/isLinXu/awesome-road-map)
- [VALSE-WorkShop](https://github.com/isLinXu/VALSE-WorkShop)
- [prompt-engineering-note](https://github.com/isLinXu/prompt-engineering-note)

## 这些文档通常承担什么作用

- 为项目保留技术上下文
- 把阅读和探索转成可复用的参考资料
- 让持续中的工作更容易回看和分享
- 展示长期学习的连续性

## 说明

一些更早期的文档仓库仍然属于公开档案的一部分，但这一页现在优先保留那些更容易和当前个人方向形成联系的内容。
