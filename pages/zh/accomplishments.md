# 亮点

这一页集中展示目前较有代表性的成果，包括研究与评测、开源工程贡献，以及公开知识整理这三条主线。

## 研究与评测成果

- MME 相关工作：参与多模态 Benchmark 与评测体系建设，是公开研究身份中的重要组成部分
- YOLO-Master 方向：和目标检测、专用 Transformer、MOE 加速等研究叙事强相关的代表性工作
- 大模型评测实践：公开工作持续把 MLLM、LLM、Benchmark 构建、评测流程和部署质量连接在一起
- 你的简历中已经体现出的量化成果，也和这条线高度一致，包括多模态评测增益、大规模图文数据构建和业务召回优化

## 开源与工程成果

- [LlamaFactory](https://github.com/isLinXu/LlamaFactory)：在高效微调、LLM、VLM 工程方向上的代表性贡献线
- [paper-list](https://github.com/isLinXu/paper-list)：当前公开主页里最强的代表仓库之一，把论文收集、自动化和公开发布流程结合起来
- [crablet](https://github.com/isLinXu/crablet)：偏向 AI Assistant 与 Agent 系统构建的实用工程项目
- [YOLOv8_Efficient](https://github.com/isLinXu/YOLOv8_Efficient)：长期可见、影响力较强的计算机视觉实用工程仓库

## 为什么这些亮点重要

这些亮点体现出来的，不只是某一个仓库、某一篇论文或某一个 Benchmark，而是你持续把下面几件事连接起来：

- 研究和评测
- 评测和部署
- 部署和可复用工程
- 工程和公开文档、公开整理

这也是这个个人主页里最稳定、最容易被看见的一条主线。
