# 亮点

这一页集中展示目前较有代表性的成果，包括研究与评测、开源工程贡献，以及公开知识整理这三条主线。

## 研究与评测成果

- MME 相关工作：Xu Lin 出现在 MME Benchmark 作者列表中，因此多模态评测是公开资料里最容易核对的一条研究线索
- YOLO-Master 方向：当前公开材料能够看出一条围绕 YOLO 检测、效率设计和结构尝试展开的研究方向
- 公开简历材料也显示了多模态评测、大规模图文数据构建和业务召回优化这几类工作

## 开源与工程成果

- [LlamaFactory](https://github.com/isLinXu/LlamaFactory)：一个公开可见的 fork 形式贡献方向，主要对应高效微调、LLM 和 VLM 工程
- [paper-list](https://github.com/isLinXu/paper-list)：一个公开可见的论文整理仓库，把收集、自动化和公开发布流程连接起来
- [crablet](https://github.com/isLinXu/crablet)：偏向 AI Assistant 与 Agent 系统构建的实用工程项目
- [YOLOv8_Efficient](https://github.com/isLinXu/YOLOv8_Efficient)：长期维护的计算机视觉实用工程仓库
- [darwin-agent](https://github.com/isLinXu/darwin-agent)、[eval-skills](https://github.com/isLinXu/eval-skills) 以及相关编码代理仓库，一起体现了当前向 Agent 系统持续延伸的方向

## 知识整理与公共影响力

- [paper-list](https://github.com/isLinXu/paper-list)：论文整理与更新系统
- [awesome-road-map](https://github.com/isLinXu/awesome-road-map)：结构化 roadmap 式公开整理项目
- [awesome-foundation-agents](https://github.com/isLinXu/awesome-foundation-agents)：和 foundation agents 相关的 awesome 风格资源合集
- [VALSE-WorkShop](https://github.com/isLinXu/VALSE-WorkShop)：围绕 workshop 与资料聚合的长期公开项目
- [paper-read-notes](https://github.com/isLinXu/paper-read-notes)、[Stanford-CS-Course](https://github.com/isLinXu/Stanford-CS-Course)、[prompt-engineering-note](https://github.com/isLinXu/prompt-engineering-note) 共同体现了长期“公开学习、公开整理、持续沉淀”的习惯

## 为什么这些亮点重要

这些亮点体现出来的，不只是某一个仓库、某一篇论文或某一个 Benchmark，而是一条反复出现的模式：

- 研究和评测
- 评测和部署
- 部署和可复用工程
- 工程和公开文档、公开整理

这也是这个个人主页里最稳定、最容易被看见的一条主线。
