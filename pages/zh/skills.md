# 技能

这一页总结的是那些在公开仓库、简历材料和技术输出中反复出现、相对更容易被验证的能力。

## 主要技术方向

- 多模态模型相关工作流
- 计算机视觉模型训练与评测
- Benchmark 与评测设计
- 数据准备与大规模数据处理
- Agent 相关工具与工程实验

## 反复出现的工程技术栈

- Python
- PyTorch
- TensorFlow
- PaddlePaddle
- OpenCV
- SQL
- Apache Spark
- C++ 与 JNI（主要出现在具体项目场景中）

## 更偏实践层的能力

- 把模糊的研究目标拆成可执行的工程任务
- 打通数据、建模、评测和部署
- 围绕工作流做可复用工具，而不是停留在单次演示
- 把仓库和笔记当作长期技术积累的一部分

## 边界说明

这一页有意保持克制，只写那些反复出现、且有公开材料支撑的能力，而不是把接触过的所有框架和概念都列出来。
